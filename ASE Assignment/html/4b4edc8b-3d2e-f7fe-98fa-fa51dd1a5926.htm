      Remarks

      Supported command types include:
      • "tri" - Creates an instance of AppTriangle.
      • "write" - Creates an instance of AppWrite.
      • "clear" - Creates an instance of AppClear.
      • "reset" - Creates an instance of AppReset.
      • "circle" - Creates an instance of AppCircle.
      • "rect" - Creates an instance of AppRect.
      • "about" - Creates an instance of AppAboutBOOSE.
      • "int" - Creates an instance of AppInt.
      • "real" - Creates an instance of AppReal.
      • "array" - Creates an instance of AppArray.
      • "while" - Creates an instance of AppWhile.
      • "for" - Creates an instance of AppFor.
      • "end" - Creates an instance of AppEnd.
      • "if" - Creates an instance of AppIf.
      • "moveto" - Creates an instance of MoveTo.
      • "drawto" - Creates an instance of DrawTo.
      • "pen" - Creates an instance of PenColour.
      • "call" - Creates an instance of Call.
      • "method" - Creates an instance of AppMethod.
      • "poke" - Creates an instance of Poke.
      • "peek" - Creates an instance of Peek.
