      Methods

      CheckCollision Checks for collisions between Lego Batman and any "heart" PictureBoxes. Removes "heart" PictureBoxes on collision and updates the score.
      Dispose Clean up any resources being used.
      (Overrides FormDispose(Boolean))
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      legobatman_MouseDown Handles the MouseDown event of the Lego Batman PictureBox. Sets up the dragging state and calculates the mouse offset.
      legobatman_MouseMove Handles the MouseMove event of the Lego Batman PictureBox. Updates the PictureBox's location based on mouse movement and checks for collisions.
      legobatman_MouseUp Handles the MouseUp event of the Lego Batman PictureBox. Stops the dragging state.
      startButton_Click Handles the Click event of the Start button. Starts the timer and enables dragging of Lego Batman.
      timer1_Tick Handles the Tick event of the timer. Increments the elapsed time every second.
